With this plugin you can remove hidden comments from your website output HTML codes.

== Description ==

This plugin will remove all kind of comments (or only your chosen list of comments) from website output HTML code, like:

`<!-- something comment by XYZ  -->`

If you dont want all comments to be automatically removed, then you can choose manual method (from Settings page).

(In case you want extended plugins, you can look through  <a href="https://wordpress.org/plugins/head-cleaner/" target="_blank">Head Cleaner</a> or similar ones.)
